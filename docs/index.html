機能

標準機能はそのままに、管理機能を追加

追加データは videos.db の新規テーブルにのみ保存。既存データに変更を加えません。

🖼️

カバー画像

動画と同名の画像をフォルダから自動割り当て。動画ページに表示。

🗂️

メタデータ拡張

評価・表示名・品番・公開日・シリーズ・メーカー・レーベル・監督・出演者・ジャンルを編集／表示。

👤

出演者一覧

氏名・ふりがな・生年月日等を管理。出演者画像、関連動画表示、重複検出に対応。

📚

シリーズ一覧

シリーズを一覧で表示。シリーズ名・本数・平均評価でソート可能。

🗓️

公開カレンダー

作品の公開日で年月別にグループ化。月／年をクリックでその条件の検索へ。

⭐

評価ソート・絞り込み

検索ページに評価ソートと ★1〜★5 の絞り込みを追加。

🔎

キーワード検索の拡張

元の検索仕様を保ったまま拡張したメタデータも検索対象に。

🏷️

出演者タグ

出演者タグをプリセットから選択。条件に応じた自動付与ルール（例：身長≧170→高身長）も設定可能。

🔍

DMM連携

品番からDMM商品情報を検索し、タイトル・出演者・カバー等を自動設定（要 API ID）。

🔖

ブックマーク

フォルダ分けで動画をブックマーク。

💾

バックアップ

このアドオンで追加したデータを JSON で書き出し／復元。動画ファイルやWomanLive標準のデータは含みません。

🧩

and more…

JSONから情報一括取り込みやUI修正など、追加機能も開発中。

検索トークン

更に快適な動画管理へ

拡張されたメタデータをクリックすると、同条件の検索を自動で実行

@maker:"名前" @series:"名前" @label:"名前" @model:"品番" @director:"名前" @genre:"名前" @performer:ID @rating:4 @releasemonth:"YYYY-MM" @releaseyear:"YYYY" @release:none @bookmark:ID

例：@series:"サンプル" と @rating:4 を組み合わせて、評価4以上の同シリーズだけを抽出。

動作要件

追加のパッケージやインストールは不要

セットアップ済みの WomanLive 同梱の部品をそのまま利用します。

セットアップ済みの WomanLive

本アドオンにWomanLive本体は含まれません。
ver.260309のWomanLiveを用意してください。

新規パッケージ不要

新しい npm パッケージは使いません。
追加のインストールは不要です。

通信は限定的

インターネット接続は、DMM商品検索とカバー画像取得のときだけです。

既存データ非破壊

追加情報は videos.db の新規テーブルに保存。
既存データは変更しません。

導入

簡単4ステップ

推奨は womanlive-ext.bat から起動する方法です。

01

ext フォルダをコピー

配布物の ext フォルダを、WomanLive の server フォルダの中へコピーします。server/ext になればOK。

02

起動用 bat を配置

womanlive-ext.bat を WomanLive のルート（womanlive.bat と同じ場所）へコピーします。

03

womanlive-ext.batから起動

WomanLive を一度終了し、womanlive-ext.bat から起動します。以後はこちらから起動してください。

04

設定する

ブラウザ右上の設定から、カバー画像フォルダや DMM API ID などを必要に応じて登録します。

⌥

別の導入方法：本体を1行だけ編集する場合

womanlive.bat のまま使いたい場合は、server/index.js の app.use(cors()); の直後に次の1行を追加します。

try { require('./ext')(app); } catch (e) { console.error('[WomanLive拡張]', e); }
FAQ

よくある質問

WomanLive 本体は書き換わりますか？

いいえ。推奨手順では --require プリロードで読み込むため、本体のファイルは一切変更しません。別手順を選んだ場合のみ、本体を1行だけ追記します。

アンインストールは簡単ですか？

はい。womanlive.bat で起動すれば拡張は無効になります。完全削除する場合は server/ext フォルダと womanlive-ext.bat を削除してください。

既存のデータは消えませんか？

追加データは新規テーブルにのみ保存され、既存データは変更しません。

DMM 連携には何が必要ですか？

各自の DMM アカウントで取得した API ID とアフィリエイト ID を設定画面に登録します。
